VS Code Beginner's Guide

This guide will get you stared with the basics of VS Code

What is VS Code?

Visual Studio Code is a powerful code editor made my Microsoft. It is an open source application which is availabe for Linux, Mac and Windows. It also has powerful features such as built in debugging, Git integration and Extensions for a varitey of languages.

How to use this guide

This guide is broken down into pieces, the side menu appears on

Open the side menu using the hamburger menu icon on the top left corner of the page

Click on each link to view more about the topic

Optionally use the bottom bar to navigate the site

The bottom bar can take you to the previous topic, home page and the next topic
