Python

You will need to have a recent version of Python3 installed on your system. Optionally install pip to manage packages. Using a virtual enviorment is recommended to install pip dependencies.

System Requiremnts

  • Python3
  • Python3-venv
  • Pip

VS Code Extensions

  • Python - Microsoft
  • Python debugger - Microsoft
  • (Optional) Jupyter - for jupyter notebook support

Virtual environment:

It is recommended to use a virtual enviorment when working with python

  • Create a new virtual env using command Python: Create enviorment
  • You can select a specific virtual environment by going to the command palette and running Python: Select Interpreter
