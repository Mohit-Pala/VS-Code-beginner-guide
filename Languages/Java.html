Java

To run java code you will need to have a recent version of JDK and JRE installed. Aditionally you will need to have them added to your path enviorment variable.

System Requiremnts

  • Recent verison of JDK and JRE, above java 8, installed and added to path.

VS Code Extensions

  • Language support for java - Redhat
  • Debugger for java - Microsoft
  • (Optional) Intellicode and Intellisense - Microsoft
  • (Optional) Extension pack for Java - Microsoft

Run and Debug

  1. Open the file containing the main class
  2. Press F5
  3. Select Java if asked
