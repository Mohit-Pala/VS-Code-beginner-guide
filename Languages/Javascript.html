Javascript

VS Code has inbuilt completion for javascript with Intellisense but to run javascript code you will need to have either a web browser or Node JS installed. You can use https://nodejs.org/en to install Node JS.

System Requiremnts

  • Node JS installed

Run and Debug

  • Press F5 and select a web browser to debug in browser
  • use terminal to run node ./{filename}.js
