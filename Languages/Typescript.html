Typescript

VS Code has inbuilt completion for typescript with Intellisense but to run typescript code you will need to have ts node installed. To do so you will have to use the npm package manager. You can get it from https://www.npmjs.com/package/ts-node.

System Requiremnts

  • Node JS installed
  • npm package manager installed
  • ts-node installed

Run and Debug

  • Use terminal to run ts-node ./{filename}.ts
