C and C++

For this guide we will use the Gnu C Compiler because it is the easiest to setup and use with VS Code. It is recommended to have GCC, G++ and GDB in your path

System Requiremnts:

  • GNU C Compiler installed
  • GCC and G++ in path
  • GDB installed
  • GNU make installed

VS Code Extensions

  • C/C++ - Microsoft
  • (Optional) C/C++ extension pack

Run and debug:

  1. Run and debug using F5
  2. Select gcc build and debug active file
