What are extensions?

If Visual Studio Code is a phone, or some device you can use, then extensions are apps you can use to increase the functionality of that device. It can a llow you to code in different languages, change the look of your setup, vie w remote machines, and so much more. The best part is, extensions are easy to download and use.

Extensions are a vital part of VS Code, no matter what you are planning on using it on. Extensions allow you to use different languages, change how your setup looks, help use remote SSH(see Remote SSH) and help use version control systems like Git(see Git). This part of the guide will show you how to install a very simple extension, “vscode-pdf”, so that you will know how to download extensions for later in this guide and beyond.

Example: Installing vscode-pdf extension

  1. First, check out the side bar on the left. Click that button that looks like four squares making a larger square, with the top right one coming off.
  2. Another sidebar should popup, that has a “installed” section and “recommended” section of extensions. Look up from there where it says “Search Extensions”.
  3. Type “vscode-pdf” without the quotation marks and press enter. A extension that look like this should appear with that exact title
  4. Click install and you are done!
