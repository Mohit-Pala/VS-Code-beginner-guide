Docker Integration

Docker engine is an OS level virtualization software. Docker is used to create reproducable containers with ease. VS Code has an extension which can manage docker containers.

System Requirements

  • Docker installed locally
  • User added to docker group to run without sudo privilages
    • to achieve this run sudo usermod -aG docker $whoami on a linux/WSL machine
  • Docker system service started
    • to achieve this run sudo systemctl enable docker.service on a machine with the systemd init system

VS Code Extensions

  • Dev Containers - Microsoft
  • Docker - Microsoft
  • (Optional) Remote Development extension pack - Microsoft

Create new docker container:

  1. Click on the docker icon in the sidebar (whale icon)
  2. In the containers subsection click the +
  3. Select container type
  4. VS Code should now create and open up the container

Open Folder in container:

  1. Open command palette
  2. Run: Dev Containers: Open Folder in Container
  3. Select the folder you want to open
  4. Select container type
  5. VS Code should now open your folder in the docker container
