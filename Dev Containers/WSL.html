WSL Integration

WSL - Windows subsystem for linux is a feature of windows 10 and above which allows you to run a linux distro on a Windows machine. It uses the Windows hyper-v virtualization to provide a seamless linux experience underneath Windows. VS Code can be used inside of WSL with very simple setup.

System Requirements:

  • Windows 10 and above
  • Windows subsystem for linux installed
  • At Least one subsystem distro installed and updated
    • sudo apt-get update for debian based distros
    • sudo dnf update for fedora based distros
    • sudo pacman -Syu for arch based distros

VS Code Extensions

  • WSL - Microsoft
  • (Optional) Remote Development extension pack - Microsoft

Opening VS Code through WSL

  1. Start your WSL distro using the start menu
  2. In the bash terminal, execute code
  3. VS Code should now open up on windows connected to WSL

Connecting to WSL through VS Code

  1. Open VS Code
  2. Open the command palette
  3. Run REMOTE_WSL:New Window
  4. VS Code should open up a new window connected to WSL
