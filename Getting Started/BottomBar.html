The Bottom Bar

Open VS Code and look at the bottom left you will see this

The x with the circle represents how many errors in your code you have.

This means code that VS Code cannot compile and will stop the code from running. The exclamation point in the triangle represents warnings, which is code that may not adhere to standards but can still run.

Open a file of code or text, and you will see "Ln 1, Col 1 Spaces: 4" the 1 and 4 will change depending on what you opened. This represents where your cursor is at any given time. The spaces represents the standard indentation.
