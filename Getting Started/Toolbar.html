Toolbar

If you open VS Code, the toolbar is the bar on the very top that looks like this. We will briefly explain what each one does in general so that you know how you can use these later on.

File

File has everything you need to deal with files, folders, and workplaces. You can make new files and folders, save them, save them as something else, open old ones, or close them.


Edit

Edit has to do with editing the specific files that you have been working on, including undoing changes, copying and pasting things, and also finding and replacing keywords.


Selection

Selection has to do with certain selections of what you are working on. You can select everything in a file, then shrink and expand it depending on what you want to do. You can also move and copy lines, as well as add your cursor in specific places.


View

View is all about how you see VS Code. You can change the appearance or the editor, and open any of the sections on the left.


Run, Terminal
