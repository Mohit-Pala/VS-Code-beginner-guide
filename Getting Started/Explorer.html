Getting Started

Looking at VS Code for the first time can be overwhelming, as it has a lot of buttons and things you can do. In order to help familiarize you with VS Code, we will show you some of the buttons and how you can use them, consisting of

  • The explorer
  • The toolbar
  • The bottom bar

We will start with the explorer.

Explorer

The explorer is something that you will use very often. First off, it contains your open editors. You can think of these like tabs in google, but they are files you have open so you can edit them. Next, you have whatever file you have opened in VS Code. You will be able to see everything inside of it, and navigate to wherever you want inside of that folder to edit it. You can also add more files or folders as you see fit.
