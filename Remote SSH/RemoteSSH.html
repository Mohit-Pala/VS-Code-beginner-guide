Remote SSH

Remote ssh allows you to connect to a remote server running a SSH server. You will be able to browse, modify and execute remote files. VS Code will install a server on the remote machine which will allow you to use most of the local features on the remote machine.

System Requiremnts:

Local

  • Openssh client

Remote

  • SSH server running
  • 1gb RAM
  • 2 core CPU

VS Code Extensions

  • Remote SSH - Microsoft
  • (Optional) Remote development - Microsoft

Connect:

  1. Open VS Code on the client
  2. Run the command Remote-SSH: Connect to host
  3. You will be prompted to add a new host if this is your first time
  4. Enter your username@server_address.com
  5. If you have already added a host, it will ask for your password
  6. Once you have entered your password you will be able to login to your server

Open remote files and folders with VS Code:

  1. Connect to Remote SSH
  2. File -> Open file
  3. FIle -> open folder
