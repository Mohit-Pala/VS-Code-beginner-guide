Terminal Basics

To access the integrated terminal in VSCode, select the ‘View’ menu and choose ‘Terminal’ from the drop down menu. The terminal can also be accessed by using the shortcut ‘Ctrl + `’ (backtick). As previously mentioned, the terminal in VSCode is integrated into the program, appearing as a panel at the bottom of the window. This means that you can work on the code and terminal at the same time without switching applications.

  1. By default, VSCode uses the system shell for the terminal (e.g., PowerShell on Windows, Bash on MacOS and Linux). However, you can switch to a different shell by clicking on the dropdown menu in the terminal panel.
  2. Navigate through directories using command-line navigation commands like ‘cd’ (change directory) and ‘ls’ (list directory). This allows you to explore the project’s file structure and execute commands in different directories.
  3. You can use the terminal to interact with Git and perform version control tasks such as committing changes, pushing and pulling from remote repositories, creating branches, and more.
